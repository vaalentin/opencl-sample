# OpenCL sample

## Requirements

- `git`
- `cmake` version `2.8` or higher.
- `gcc` or `clang` that supports `std=c++11`.
- `OpenCL`

## Installation

```
$ git clone git@github.com:vaalentin/opencl.git
$ cd opencl
$ mkdir build
$ cd build
$ cmake ..
```

## Usage

```
$ cd build
$ make run
```
